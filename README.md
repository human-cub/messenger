# Мессенджер

Лёгкий веб‑мессенджер на vanilla JS. Цель — общаться с РФ через Chrome на старом iPad mini 2/3 (iOS 12), без блокировок и без больших фреймворков.

## Что внутри

- **Один HTML + один JS** — никаких сборщиков. ~1500 строк всего.
- **Supabase** (Postgres + Auth + Realtime + Storage) — бэкенд. Адрес уже зашит, проект `messenger` в регионе Frankfurt (eu‑central‑1) — близко к РФ, обычно доступен без VPN.
- **WebRTC** — аудио/видеозвонки P2P, signaling через Supabase Realtime, STUN/TURN бесплатные.
- **Голосовые** — пишутся через WebAudio + кодирование в WAV вручную (MediaRecorder в iOS 12 нет).
- **UI как в Telegram** — слева контакты, справа чат, скрепка/микрофон/самолёт снизу, трубка/камера сверху.

## Перед первым запуском — один обязательный шаг

В Supabase нужно **отключить подтверждение email**, потому что мы регистрируем юзеров по нику, а email там фейковый (`username@msg.local`).

1. Открой <https://supabase.com/dashboard/project/rfqdlgaxtlkhlyabwayx/auth/providers>
2. Нажми на провайдер **Email**.
3. Выключи галочку **Confirm email**.
4. Save.

Это надо сделать один раз.

## Деплой

Сайт нужно открыть **по HTTPS** — иначе iOS не даст микрофон/камеру и WebRTC. Открыть `index.html` с iPad напрямую (`file://`) тоже нельзя.

Самый быстрый способ — **Cloudflare Pages** (доступен в РФ, бесплатно, HTTPS из коробки):

1. Зарегистрируйся на <https://dash.cloudflare.com/sign-up>.
2. Pages → Create a project → Direct Upload.
3. Закинь zip с `index.html` и `app.js`. Project name — любой.
4. Через минуту получишь адрес `имя.pages.dev` — открывай на iPad.

Альтернативы: **Netlify Drop** (просто перетащи папку на <https://app.netlify.com/drop>), **GitHub Pages** (если уже есть git‑репо).

## Как пользоваться

1. Заходишь на сайт с двух разных устройств/браузеров.
2. На каждом — регистрируешь свой ник + пароль.
3. На одном жмёшь «**+**» сверху слева → пишешь ник второго → **Добавить**. Контакт автоматически появится у обоих.
4. Открываешь чат → пишешь, шлёшь фото/видео/голосовые, звонишь.

## Особенности и ограничения iPad mini 2/3 (iOS 12)

| Фича | Как работает |
|---|---|
| Текстовые сообщения | OK |
| Фото и видео из галереи | OK (до 20 МБ) |
| Голосовые | OK, формат WAV mono 16 кГц (~30 КБ/сек) |
| Аудиозвонок | OK |
| Видеозвонок | Работает, но видео может тормозить — это железо |
| Уведомления при открытой вкладке | Звук + бейдж в title + красная точка на favicon |
| Уведомления при закрытом браузере | **Невозможны** на iOS до 16.4. Web Push в Safari появился только на новых устройствах. |

Если оставлять вкладку открытой — звук + индикатор в title работают и привлекут внимание.

## Если Supabase заблокируют в РФ

Фронт отвязан от бэкенда — переезд занимает час:

1. Поднимаешь Postgres где угодно (Render, Railway, свой VPS, Yandex Cloud).
2. Накатываешь миграции из `supabase/migrations.sql` (см. ниже).
3. Поднимаешь свой signaling/realtime — проще всего поставить self‑hosted Supabase или заменить на свой WebSocket‑сервер.
4. В `app.js` меняешь `SUPABASE_URL` и `SUPABASE_ANON_KEY` — всё.

## Известные тонкости

- Бесплатный публичный TURN (`openrelay.metered.ca`) имеет лимиты по трафику. Если звонки начнут падать — заведи свой TURN или подключи Cloudflare Calls / Twilio Network Traversal.
- Supabase Free: 500 МБ БД, 1 ГБ хранилища, 50 тыс. MAU. На двоих‑троих хватит надолго.
- На iPad **обязательно** добавь сайт на главный экран (Поделиться → На главный экран) — будет как нативное приложение без баров.
